New York City Property Sales Data Analysis

This project was part of Studi Independen Program from ministry of Education. The project focusses on analyze data of New York from more than 84K property data. You can see the data here


NYC Property Sales shows total residential units are more profitable than commercial units. However, event though residential unis more profitable, their sales persentage has decreased.


From the Problem described above, we can conclude that this project objective is to increase residential unit sales by 5% for September 2017 to August 2018. With data scope created from Semptember 2016 to August 2017.


After Analyze the data, I start to design a dashboard to show the comparison from several variables. I use Looker Studio for the dashboard, considering it give free option and support for data import from Google Spreadsheet. Make it More Efficient to share the result with others.
